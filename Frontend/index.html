Electiva — Frontend

Aplicación frontend mínima. Coloca aquí tus ficheros HTML/CSS/JS o crea una app con el framework que prefieras.

Paso rápido

  1. Editar este archivo o añadir `index.js` y `styles.css` en esta carpeta.
  2. Servir estáticamente: npx serve Frontend o con tu servidor favorito.
  3. Si vas a usar un framework (React/Vue/Angular), crea la estructura del proyecto dentro de `Frontend/`.
